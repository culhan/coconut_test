## Installation

Via Composer

``` bash
$ composer require khancode/laravelbaserest
```

migrate

```
$ php artisan migrate
```

seed

```
$ php artisan db:seed
```

## Usage

for admin login with

```
email = admin@admin.com
password = password
```

for user login with

```
email = user@user.com
password = password
```
